快速上手 #

本节将介绍如何在项目中使用 Howbuy OtcUI

注意 #

Howbuy OtcUI 是一个基于 ElementUI 进行二次封装的组件库, 所以在使用的时候，别忘了安装注册ElementPlus组件库。

完整引入 #

如果你对打包后的文件大小不是很在乎，那么使用完整导入会更方便。

在项目的 main.ts 入口文中

import { createApp } from 'vue'

import ElementPlus from 'element-plus'
import HowbuyOtcUI from '@howbuy/otc-ui'

import 'element-plus/dist/index.css'
import '@howbuy/otc-ui/dist/index.css'

import App from './App.vue'

const app = createApp(App)

app.use(ElementPlus)
app.use(HowbuyOtcUI)
app.mount('#app')

按需导入 #

您需要使用额外的插件来导入要使用的组件。

<template>
  <OtcButton>I am OtcButton</OtcButton>
</template>
<script>
  import { ElButton } from 'element-plus'
  import { OtcButton } from '@howbuy/otc-ui'

  export default {
    components: { ElButton, OtcButton },
  }
</script>
