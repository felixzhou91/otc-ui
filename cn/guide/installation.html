介绍&安装 #

介绍 #

Howbuy OtcUI是一个基于 ElementPlus 进行二次封装的组件库，方便快速开发Howbuy风格的后台系统。

安装 #

1. 切换 npm 私服，shell 终端执行 #

npm config set registry=http://npm.howbuy.pa/

2. 使用包管理器 #

# 选择一个你喜欢的包管理器

# NPM
$ npm install @howbuy/otc-ui --save

# Yarn
$ yarn add @howbuy/otc-ui

# pnpm
$ pnpm install @howbuy/otc-ui

3. 或者浏览器直接引入 #

直接通过浏览器的 HTML 标签导入 Howbuy OtcUI，然后就可以使用全局变量 HowbuyOtcUI 了。

<head>
    <!-- vue3.x -->
    <script src="https://unpkg.com/vue@next"></script>

    <!-- Element Plus 组件库&样式 -->
    <link rel="stylesheet" href="https://unpkg.com/element-plus/dist/index.css">
    <script src="https://unpkg.com/element-plus"></script>

    <!-- Howbuy OtcUI 组件库&样式 -->
    <link rel="stylesheet" href="./dist/index.css">
    <script src="./dist/index.full.js"></script>
</head>
<div id="app">
    <OtcButton type="primary">测试2</OtcButton>
</div>

<script>
const App = {
    data() {
        return {
        message: "Hello OTC UI",
        };
    },
};
const app = Vue.createApp(App);

app.use(ElementPlus);
app.use(HowbuyOtcUI);

app.mount("#app");
</script>
