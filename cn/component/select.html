Select 下拉选择器 #

当选项过多时，使用下拉菜单展示并选择内容。

禁用状态 #

禁用整个选择器组件

可清空单选 #

您可以使用清除图标来清除选择。

显示选项的值 #

多选 #

多选选择器使用 tag 组件来展示已选中的选项。

自定义模板 #

你可以自定义如何来渲染每一个选项。

将选项进行分组 #

你可以为选项进行分组来区分不同的选项

筛选选项 #

可以利用筛选功能快速查找选项。

远程搜索 #

输入关键字以从远程服务器中查找数据。

使用值键 value-key 属性 #

如果 Select 的绑定值为对象类型，请务必指定 value-key 作为它的唯一性标识。
