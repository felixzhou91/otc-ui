FormTable 表格表单 #

可编辑的表格，带表单校验。

数据录入（列表行） #

比较常用的一个用法，列表型表单行数据的录入，可以新增删除行数据。

拖拽 & 禁用 #

设置属性 isDrag 可添加拖拽功能，dragDisabled 属性可控制拖拽是否可用。disabled 属性则禁用整个表格的操作。

类型 #

FormTable 配置项类型。

// OtcTableList 表格列类型
export type TableColumnType = {
    key: string // 字段名称
    label: string // 列中文名
    minWidth?: string // 最小宽度
    width?: string // 宽度
    className?: string // 类名
    isHidden?: boolean // 是否隐藏
    isSlot?: boolean // td 是否使用slot
    isSlotHeader?: boolean // th 是否使用slot
    showOverflowTooltip?: boolean // 超出部分隐藏
    fixed?: 'left' | 'right' // 固定列位置 left ｜ right，
    align?: 'left' | 'center' | 'right' // 居中位置
    formatter?: (row: any, column?: any, value?: any) => any // formatter方法
}
export type FormTableColumnType = TableColumnType & {
    require?: boolean // 是否显示必填（表头显示星号）
    sortable?: string // 列排序
    rules?(scope?: any, column?: any, params?: any): any // 字段校验规则
}
